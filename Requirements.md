Requirements

Tools Used:

Language: Java 11 or later

IDE: Eclipse or IntelliJ IDEA

Concurrency: Java Thread Pool Executors (ExecutorService, CountDownLatch, Semaphore)

Randomization: java.util.Random for initial battery levels, arrival times, and task generation

Logging: Existing Logger.java system extended with warn() support

Testing Framework: JUnit 5

Documentation: Markdown for README and reports
